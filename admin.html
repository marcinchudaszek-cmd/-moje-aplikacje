GitHub / Publikacja

Skonfiguruj połączenie z GitHub żeby publikować zmiany jednym kliknięciem.

Ustawienia GitHub
Stwórz token na github.com/settings/tokens/new — zaznacz tylko "repo"
Jak to działa?
1️⃣ Wpisz token GitHub (jednorazowo)
2️⃣ Edytuj treści w panelu (aplikacje, profil, kontakt...)
3️⃣ Kliknij "Publikuj zmiany" na górze
4️⃣ Po ~1 minucie zmiany widoczne na beagleapps.pl!

Gdzie zapisują się dane?

Dane zapisują się w localStorage przeglądarki.

  • Dane zostają po zamknięciu przeglądarki ✅
  • Dane są tylko w TWOJEJ przeglądarce ✅
  • Użyj "Eksportuj" żeby zrobić backup 💾
